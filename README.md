# MyGarage
Use MyGarage to keep track of the vehicles in your garage, consult periodically on how many miles each vehicle has, be alerted for your car service on each one every 100000km.

This app uses : API for the logo of the cars and WorkManager for car service notifications.

This app is tested on 2 device a Nothing phone(1) with a display of 6,55 inches and a tablet of 10 inches for each one of them we have a portrait layout and a landscape one.
